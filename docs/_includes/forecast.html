How much does adding Trends data outperform models not using it?

The most efficient model we found is the GARCH(1,1) with the selected Trends data as mean regressors and Student's t distributed errors. So the same model but we just add trends data as predictors in the mean process.
The formulas of the model are the following:

\[r_t = \mu + \text{dow jones.nsa}_t + \text{crisis.nsa}_t + \epsilon_t\] \[\sigma^2_t = \omega + \alpha \epsilon_{t-1}^2 + \beta \sigma_{t-1}^2\] \[\epsilon_t = \sigma_t e_t,\,\,\, e_t \sim t_{\nu}\]

Fitting the model to the data, we got the following results:
